# Holistic-GDNF-Stimulation-ASD-Theory
Building on established evidence that GDNF promotes the survival/recovery of dopaminergic neurons in Parkinson’s Disease (Lin et al., 1993; Gash et al., 1996), this research posits that these neurorestorative benefits extend to neurodevelopmental disorders, such as Autism Spectrum Disorder (ASD), which are characterized by brain wiring anomalies.
